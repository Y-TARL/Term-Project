# :mortar_board: 浙江工业大学计算机科学与技术专业课程设计合集

本仓库收录了浙江工业大学计算机科学与技术专业（科学方向）本科阶段的所有课程设计和大作业项目。

## :books: 课程列表

### 核心课程
- **操作系统** - Pintos操作系统内核实现
- **计算机组成原理** - 汇编程序设计与体系结构
- **计算机网络** - 网络协议实现与设备配置
- **数据库系统** - PostgreSQL数据库设计
- **数据结构** - 算法实现与应用

### 编程课程
- **C++程序设计** - 面向对象编程实践
- **软件工程** - 全栈Web应用开发
- **人工智能导论** - AI算法与机器学习

### 硬件课程
- **数字逻辑电路** - FPGA硬件设计
- **电路与电子技术基础** - 模拟电路设计

### 综合实践
- **计算机工程实践** - 智能机器人系统集成

## :book: 使用说明

每个课程文件夹包含完整的项目代码、文档和报告，供学习参考使用。
