# 数据库系统课程设计源代码说明

## 项目概述
本项目是一个基于PostgreSQL的大学生成绩管理系统，实现了完整的学生、课程、成绩管理功能。

## 技术栈
- **后端**: Java + Servlet + JDBC
- **数据库**: PostgreSQL 
- **前端**: JSP + HTML + CSS + JavaScript
- **服务器**: Tomcat

## 主要功能
1. **学生管理**: 学生信息的增删改查
2. **课程管理**: 课程信息维护
3. **成绩管理**: 成绩录入、查询、统计
4. **数据分析**: 成绩分布、排名统计
5. **权限管理**: 管理员和普通用户权限分离

## 数据库设计
- 省份表 (Province)
- 城市表 (City) 
- 区县表 (District)
- 专业表 (Majors)
- 班级表 (Classes)
- 课程表 (Courses)
- 学生表 (Students)
- 成绩表 (Scores)

由于隐私保护，原始源代码中的个人标识符已被移除或替换。如需完整源代码，请参考SQL文件重新生成对应的Java实体类和DAO层代码。

## 部署说明
1. 创建PostgreSQL数据库
2. 执行SQL脚本建表和插入测试数据
3. 修改数据库连接配置
4. 部署到Tomcat服务器
